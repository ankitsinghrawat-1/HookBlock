# HookBlock 🛡️
**A Proactive Phishing Detection Browser Extension**

HookBlock is a security tool designed to identify and alert users about potential phishing links in real-time. It uses pattern matching and security heuristics to ensure a safer browsing experience.

### 🚀 Features
* **Real-time Scanning:** Analyzes URLs as they load.
* **Security Alerts:** Immediate visual warnings when a suspicious link is detected.
* **Lightweight:** Minimal impact on browser performance.

### 🛠️ Tech Stack
* **Languages:** JavaScript, HTML, CSS
* **APIs:** Chrome Extension API (Manifest V3)
* **Security Logic:** Custom heuristic-based filtering.

### ⚙️ Installation
1. Clone the repo: `git clone https://github.com/ankitsinghrawat-1/hookblock.git`
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select the project folder.
